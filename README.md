# 阳光颜(盐)语 · 英语词汇量在线测试

基于语言学家 **Paul Nation** 开发的 **14 Thousand** 词汇量测试工具，专为辅导孩子英语学习而设计的在线测试系统。

## 关于 14 Thousand 测试

该工具是基于语料库和词频统计的，**1 个词可以代表 100 个词**，而且是 **word family**，不是单个的词。

> word family 指一个基础词的各种变化，比如一个动词，它的过去时、过去分词、现在分词、加 -tion 词缀变成名词，加 -ful 词缀变成形容词等等，所有这些变化，算成一个 **word family**，而不是计算成单个的单词。
>
> 以 word family 计数的词汇测试，算出来以后，都会觉得挺少的。如果每个单词只要不一样就算一个词，那会翻倍。比如说，前 1000 个最常用的单词 word family，其实有 4000 不同形式。

## 功能特性

- **14 组逐级测试**：每组 10 题，覆盖 First 1000 到 Fourteenth 1000
- **分级进度导航**：14 个阶段按键，绿色已完、灰色未做，可任意跳转重做
- **自动估算词汇量**：每题代表 100 个词族，总词汇量 = 答对总数 × 100
- **错题本**：记录所有答错的题目，重做后做对的自动移除
- **语音朗读**：点击喇叭按钮，使用 Karen / Samantha 等真人语音朗读单词和例句
- **管理员后台**：学生增删改查，查看成绩详情和错题统计

## 快速开始

```bash
pip install flask flask-sqlalchemy werkzeug
python3 app.py
```

访问 `http://localhost:5050`

### 学生使用

首页输入姓名和角色（如班级），即可从第 1 组开始逐级测试。

### 管理员登录

访问 `/admin/login`

- 默认账号：`admin`
- 默认密码：`admin123`（可通过环境变量 `ADMIN_PASSWORD` 修改）

## 技术栈

- **后端**：Python Flask + SQLite
- **前端**：原生 HTML + CSS（儿童友好界面）
- **语音**：Web Speech API（SpeechSynthesis）
